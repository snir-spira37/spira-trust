## Scope

This authorization permits only a narrow DS-R0 runner fix:

```text
read the frozen schema file
parse it as JSON
serialize the parsed object as deterministic inline canonical JSON
pass that inline JSON string to --json-schema
record schema transport and schema identity metadata
add focused tests
update DS-R0 runner-fix report/results
```

The fix must not change the schema semantics.

## Required Proof

The fix must report:

```text
schema_transport: INLINE_CANONICAL_JSON
schema_semantics_changed: false
schema_file_sha256: unchanged relative to current HEAD
inline_schema_sha256: SHA256(canonical compact JSON of parsed schema object)
parsed_schema_object_matches_file: true
additionalProperties: unchanged
required fields: unchanged
enum/const constraints: unchanged
```

Canonical inline JSON must be generated by JSON parsing and deterministic JSON
serialization, not by string replacement.


ready for a separately authorized full DS-R0 rerun from P1.
