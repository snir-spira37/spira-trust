## Authorized Claim

SPIRA validates structure, binding, evidence integrity, and safe evidence paths
for `SEVERANCE_AUTHORIZATION` and `LEGACY_ISOLATION_RESULT`.

## Boundary

Phase 1 does not validate cryptographic signature trust, signer identity or
authority, actual isolation execution, or permission to sever.

This package prepares external reproduction. It is not itself an external
reproduction result.
