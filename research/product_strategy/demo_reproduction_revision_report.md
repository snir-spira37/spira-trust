## Root Cause

The mismatched artifacts were:

```text
research/formal_core/domain3/spira_formal_core_v1_domain3_raw_adapter_conformance_report.md
research/formal_core/domain3/spira_formal_core_v1_domain3_raw_adapter_conformance_results.json
research/formal_core/domain3/spira_formal_core_v1_domain3_raw_adapter_conformance_review.md
```

The artifacts were generated by the Domain 3 raw-adapter conformance harness. The old result format stored raw pytest output tails, including duration text. On Windows, the harness also wrote generated files with CRLF working-tree bytes while the repository normalizes text to LF. Together, those made the generated files byte-unstable across otherwise equivalent runs. The package hash layer correctly detected the drift.
